# Futurethinking-web-2026

This is the official repository for the Futurethinking-web-2026 project.

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install` (in Frontend/Backend directories as needed).
3.  Set up environment variables (see `.env.example` if available).
4.  Run the development server.

## Build

To build the project:
```bash
cd Frontend
npm run build
```
The output will be in `dist_final`.
